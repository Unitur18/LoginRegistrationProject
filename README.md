# Login & Registration System

## Overview

This project is a web-based user authentication system developed using Laravel. It provides essential user management features, including account registration, login, session management, and secure authentication.

The project was developed to gain practical experience with Laravel's MVC architecture, routing system, controllers, Blade templates, and database integration.

## Features

* User Registration
* User Login
* Authentication System
* Session Management
* Form Validation
* Password Security
* Database Integration
* MVC Architecture

## Technologies Used

* PHP
* Laravel 10
* Blade Template Engine
* MySQL
* HTML
* CSS
* JavaScript

## Project Structure

* **Routes:** Handles application endpoints.
* **Controllers:** Manages authentication logic.
* **Views:** User interface built with Blade templates.
* **Database:** Stores user information securely.
* **Middleware:** Controls authentication and access management.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Unitur18/LoginRegistrationProject.git
```

2. Install dependencies:

```bash
composer install
npm install
```

3. Create environment file:

```bash
cp .env.example .env
```

4. Generate application key:

```bash
php artisan key:generate
```

5. Configure database settings in `.env`

6. Run migrations:

```bash
php artisan migrate
```

7. Start the development server:

```bash
php artisan serve
```
